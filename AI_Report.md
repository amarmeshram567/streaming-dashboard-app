## 1. Project Objective

The Streaming Dashboard Clone is a simplified streaming service dashboard inspired by platforms like Netflix and Hulu.  
The app features:

- **Hero Banner:** Displays featured content prominently.  
- **Horizontal Movie Rows:** Scrollable rows for popular, trending, or categorized movies.  
- **Movie Detail Pages:** Detailed view for each movie with title, overview, release date, and genres.  
- **Search Functionality:** Users can search movies directly in the app without leaving the page.  

The main goal was to build a fully responsive, performant, and deployable Next.js 14 + TypeScript application using Tailwind CSS.
